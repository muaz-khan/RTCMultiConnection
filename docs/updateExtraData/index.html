RTCMultiConnection.updateExtraData | "method"

HOME © Muaz Khan . @WebRTCWeb . Github . Latest issues . What's New?

"connection.updateExtraData" method allows you update current user's extra data on socket.io server. It will auto-sync extra-data among all connected users sides.

"updateExtraData" method requires RTCMultiConnection >= v3.

"connection.updateExtraData" method takes nothing; returns nothing: 

connection.updateExtraData();

"connection.onExtraDataUpdated" event will be fired on all other users' sides: 

connection.onExtraDataUpdated = function(event) {
    var extra = event.extra;
    var targetUserId = event.userid;
};
